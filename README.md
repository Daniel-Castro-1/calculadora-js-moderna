# 🧮 Calculadora Interativa em JavaScript

Esta é uma calculadora moderna e funcional desenvolvida para praticar os fundamentos do desenvolvimento Web: **HTML5, CSS3 e JavaScript**. O projeto foca na experiência do usuário (UX), atualizando os resultados em tempo real à medida que os valores são inseridos.

## 🚀 Funcionalidades

- **Cálculos em Tempo Real:** Não é necessário clicar num botão para ver o resultado; ele aparece instantaneamente ao digitar ou mudar a operação.
- **Quatro Operações Básicas:** Adição, Subtração, Multiplicação e Divisão.
- **Tratamento de Dados:** Conversão de strings para números para garantir cálculos precisos.
- **Design Moderno:** Interface estilizada com CSS, utilizando sombras, bordas arredondadas e uma paleta de cores vibrante.

## 🛠️ Tecnologias Utilizadas

- **HTML5:** Estrutura semântica e formulários.
- **CSS3:** Estilização, Flexbox para alinhamento e design responsivo.
- **JavaScript (ES6+):** Manipulação do DOM, `addEventListener` (eventos de `input` e `change`) e lógica condicional.

## 💡 O que eu aprendi neste projeto

Neste projeto, superei desafios fundamentais de programação:
1. **Prevenção de Erros de Tipagem:** Aprendi a usar `Number()` para evitar que `1 + 1` resultasse em `11`.
2. **Escopo de Variáveis:** Entendi a diferença entre capturar um elemento e capturar o seu valor no momento certo dentro de uma função.
3. **Event Listeners:** Implementei "ouvintes" para tornar a página dinâmica e reativa às ações do usuário.
